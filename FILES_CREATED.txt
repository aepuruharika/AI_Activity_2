🏗️ ARCHITECTURE OVERVIEW
════════════════════════════════════════════════════════════════════════════════

Resume + JD Input (React Form)
    ↓
FastAPI Backend (main.py)
    ├─ Step 1: llm_resume_extractor.py (Llama)
    │   → Extract: name, skills, experience, strengths, education
    │   → Output: Structured JSON
    │
    ├─ Step 2: llm_matcher_scorer.py (Mistral)
    │   → Score resume vs JD (0-100%)
    │   → Logic:
    │       • If score > 70%: Generate 5 interview questions
    │       • If score ≤ 70%: Generate rejection reasons + improvements
    │   → Output: Score + Questions or Reasons
    │
    └─ Step 3: llm_recruiter_summary.py (Zephyr)
        → Generate executive summary
        → Recommendation + next steps
        → Interview complexity level
        → Output: Recruiter summary

React Frontend (Results Display)
    → Shows all results in beautiful UI with animations

🚀 QUICK START
════════════════════════════════════════════════════════════════════════════════

1. Get HuggingFace API Key:
   → Visit: huggingface.co/settings/tokens
   → Create new token (Read access)

2. Setup Backend:
   cp .env.example .env
   # Edit .env with your API key
   python -m venv venv
   venv\Scripts\activate  # Windows
   pip install -r requirements.txt

3. Setup Frontend:
   cd frontend
   npm install

4. Start Services:
   Terminal 1: python main.py
   Terminal 2: cd frontend && npm run dev

5. Open Browser:
   http://localhost:5173

6. Test:
   • Enter job title
   • Upload resume
   • Paste job description
   • Click "Analyze Resume"
   • See results!
